# MFPS

Перемещается байт из PSW в операнд.
Операнд рассматривается, как байтовый (т.е. адрес может быть нечетным)

Пример: MFPS ОП

Результат: ОП := PSW

| Флаг | Значение                              |
|------|---------------------------------------|
| N    | 1, если бит 7 в PSW равен 1, иначе 0  |
| Z    | 1, если младший байт PSW = 0, иначе 0 |
| V    | 0                                     |
| C    | Не изменяется                         |
