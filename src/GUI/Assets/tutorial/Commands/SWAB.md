# SWAB

Меняет байты слова местами

Пример: SWAB ОП

Результат: Байт1 - Байт0 := Байт0 - Байт1

| Флаг | Значение                                         |
|------|--------------------------------------------------|
| N    | Равен биту 7 результата                          |
| Z    | 1, если младший байт результата равен 0, иначе 0 |
| V    | 0                                                |
| C    | 0                                                |
