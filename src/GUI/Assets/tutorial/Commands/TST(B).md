# TST(B)

Устанавливает коды условий N и Z в соответствии с значением операнда

Пример: TST ОП

| Флаг | Значение                               |
|------|----------------------------------------|
| N    | 1, если опреанд отрицательный, иначе 0 |
| Z    | 1, если операнд равен 0, иначе 0       |
| V    | 0                                      |
| C    | 0                                      |
