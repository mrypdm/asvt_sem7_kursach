# BIT(B)

Проверяет биты

Пример: BIT ОП1, ОП2

Результат: ОП1 & ОП2

| Флаг | Значение                                 |
|------|------------------------------------------|
| N    | 1, если результат отрицательный, иначе 0 |
| Z    | 1, если результат 0, иначе 0             |
| V    | 0                                        |
| C    | Не изменяется                            |
