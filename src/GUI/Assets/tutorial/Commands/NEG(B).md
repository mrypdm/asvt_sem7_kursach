# NEG(B)

Содержимое указанного операнда дополняется до двух

Пример: INC ОП

Результат: ОП := -ОП

| Флаг | Значение                                 |
|------|------------------------------------------|
| N    | 1, если результат отрицательный, иначе 0 |
| Z    | 1, если результат равен 0, иначе 0       |
| V    | 1, если операнд равен 100000, иначе 0    |
| C    | 0, если резульат равен 0, иначе 1        |
