# CMP(B)

Сравнивает два операнда

Пример: CMP ОП1, ОП2

Результат: ОП1 - ОП2

| Флаг | Значение                                                                                    |
|------|---------------------------------------------------------------------------------------------|
| N    | 1, если результат < 0, иначе 0                                                              |
| Z    | 1, если результат = 0, иначе 0                                                              |
| V    | 1, если операнды разного знака и знак результата равен результату второго операнда, иначе 0 |
| C    | 0, если возник перенос                                                                      |
