# COM(B)

Содержимое указанного операнда инвертируется побитово

Пример: COM ОП

Результат: ОП := ~ОП

| Флаг | Значение                                        |
|------|-------------------------------------------------|
| N    | 1, если старший бит результата равен 1, иначе 0 |
| Z    | 1, если результат равен 0, иначе 0              |
| V    | 0                                               |
| C    | 1                                               |
